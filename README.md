# GLOBAL-EARTHQUAKE-DASHBOARD-1900-2025-


**Overview**

An end-to-end data analytics and visualization project analyzing historical global earthquake data (1900–Present).
The project focuses on identifying seismic patterns, regional risk, and temporal trends using Python for data preparation and Tableau for interactive dashboards.



**Business Questions Answered**

Where do the strongest earthquakes occur globally?

How has earthquake activity changed over time?

Which regions experience higher average magnitudes?

Is there a relationship between earthquake depth and magnitude?

Do earthquakes follow specific time-of-day patterns?


**Dataset**

Source: Significant Earthquakes (USGS-based historical records)

Records: Global seismic events from 1900 onward

Key Variables: Date, Location, Magnitude, Depth, Region


**Data Preparation (Python)**

Data was cleaned and transformed using Pandas to ensure analysis-ready structure.

**Key steps**:

Datetime parsing and feature extraction (year, hour)

Data type correction for magnitude and depth

Handling missing and inconsistent values

Exporting a clean dataset for visualization using python


**Dashboard Highlights (Tableau)**

The interactive dashboard includes:

Global Map: Earthquake hotspots by magnitude

Trend Analysis: Earthquakes per year & average magnitude

Distributions: Magnitude and depth histograms

Regional Analysis: Average magnitude by region

Time Heatmap: Earthquake activity by hour and day

Scatter Analysis: Depth vs magnitude

Each chart is fully interactive and connected via filters for deeper exploration.


**Key Insights**
Earthquakes cluster strongly along tectonic plate boundaries

Most significant events fall within magnitude 5–7

High-magnitude earthquakes are rare but regionally concentrated

Shallow earthquakes dominate the dataset

Some regions show fewer events but higher average intensity

No consistent upward trend in earthquake strength over time
